# Typo Effects Studio

Single-page typographic asset generator for procedural text patterns, live animated previews, and web-ready exports.

## Features

- Editable spiral, field, and tunnel text patterns
- Split preview across multiple canvas sizes
- Loop presets for animated web assets
- Export to SVG, PNG, JPG, APNG, GIF, Lottie, and MP4
- Project-local fonts and outlined vector exports for SVG/Lottie
